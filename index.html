Miscellaneous Qlik Development Tools

Curated By Phuoc Tran Minh

Bookmarklets (simply drag the link to your browser bookmarks to save&use)

  • Fount - Enable/Disable Font Checking Mode, source and more info: artequalswork.com
  • Qlik Dev Menu - add "Developer" option in context menu to quickly check object id, type and other properties.
  • QMC Apps - Open QMC Apps of whatever Qlik server you're currently using in new window.
  • Qlik New App Session - Opens an alternate Qlik session that has independent selections for custom comparisons.
